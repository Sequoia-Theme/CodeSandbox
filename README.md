# Sequoia for Linear App
Sequoia theme adapted to Code Sanbox https://codesandbox.io/


# Installation

- Open VS Code
- Press (CMD/CTRL) + Shift + P
- Enter 'Developer: Generate Color Theme From Current Settings'
- Copy the contents
- Paste them in CodeSandbox Preferences

- Monochrome:
